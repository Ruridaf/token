TOKEN


Parte integrante do ecossistema blockchain, com o surgimento do Ethereum em 2015 o padrão ERC-20 passa a permitir a criação de tokens para contratos inteligentes.

Nota: O Chainlink não é um aplicativo descentralizado com uma interface de utilizador como a Uniswap por exemplo (será mais uma rede descentralizada), mas eu acho o projeto interessante e resolvi escrever sobre ele.


LINK

  • A história do Chainlink remonta a setembro de 2014, quando Steve Ellis, Ari Juels e Sergey Nazarov iniciaram o projeto SmartContract.com
  • Três anos depois, em setembro de 2017, lançaram o whitepaper Chainlink
  • Os investidores que acreditaram no projeto injetaram 32 milhões de dólares de financiamento e em junho de 2019, a rede principal Chainlink foi lançada com sucesso

Características e funcionalidades

  • Chainlink serve como ponte para que diferentes blockchains e contratos inteligentes comuniquem com dados do mundo real.
  • Oferece diversos serviços, como Data Feeds ou VRF (Verifiable Random Function).
  • Usa uma rede de nós (oracles) para garantir a integridade dos dados fornecidos, tornando-os confiáveis e verificáveis.
  • LINK é um utility token usado para pagar pelos serviços na rede Chainlink.
  • É um token ERC-20 baseado na blockchain Ethereum.
  • Tambem é usado como incentivo aos operadores de nós (oracles) para fornecerem dados precisos e confiáveis.

Uma mineração diferente

  • LINK é criado e emitido através de contratos inteligentes na blockchain Ethereum.
  • Foram criados Mil Milhões de tokens.
  • A distribuição inicial foi através de uma ICO (Initial Coin Offering) em 2017.
  • A distribuição ocorre através de vendas, recompensas e programas de incentivo.
  • Por exemplo os operadores de nós (oracles) são incentivados com LINK para fornecer dados corretos, mas podem ser penalizados (com perda de tokens) se fornecerem dados incorretos.
  • Atualmente, cerca de 608 milhões de tokens estão em circulação.
